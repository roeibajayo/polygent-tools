# Build Project

This is a **task step** that executes a script or command.

Task steps support:
- Different script types (auto, bash, powershell)
- Automatic advancement on completion
- Failure handling with custom messages
- Output capture and display

```bash
# Example build command
echo "Building project..."
sleep 2
echo "Build completed successfully!"
```

Task steps are ideal for:
- Running builds
- Executing tests
- Deploying applications
- Running any automated commands
